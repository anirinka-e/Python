# Python

## Шаблон для автоматизации тестирования

### Шаги
1. Склонировать проект 'git clone '
2. Установить зависимости
3. Запустить тесты 'pytest'

### Стек:
- pytest
- selenium
- requests
- allure
- config

### Структура
- ./test - тесты
- ./pages - описание страниц
- ./api - хелперы для работы с API
